# Story Time — The Anthropic Reading Room

Long essays and dense papers on AI — the safety case, the science, and the future worth fighting
for — turned into short, **narrated, scroll-driven listening experiences**.

**▶ Live:** https://eric-sabe.github.io/story-time/

Each piece becomes a full-screen reading with real voiceover (one audio clip per scene, with
word-synced captions), a palette that shifts by section, count-up statistics, and a floating player
with playback-speed control. Browse the library by **Shelves** (themed) or **Chronologically**, and
every card links to the original source.

*Designed by Opus · Narrated by ElevenLabs.*

## The library

Grouped into shelves: **The Safety Case · Reading the Machine · When Models Misbehave · Teaching
Values · The Future & The Stakes · The People** — spanning Anthropic's *Core Views on AI Safety*,
Dario Amodei's *Machines of Loving Grace*, *Constitutional AI*, *Claude's Constitution*, the
interpretability work (*Scaling Monosemanticity*, *The Biology of an LLM*), the alignment-failure
research (*Alignment Faking*, *Sleeper Agents*, *Agentic Misalignment*), and more.

## How it's built

Everything is generated by [`build.mjs`](build.mjs) from per-work data files in `works/`. A single
source of truth per work holds both the on-screen copy and the spoken narration, so text, audio, and
captions never drift. See [APPROACH.md](APPROACH.md) for the full architecture.

New works are added through a Claude Code pipeline (`/add-work`, `/add-works`) that distills a source,
then runs it past an **independent adversarial fidelity reviewer** — every spoken and on-screen claim
is checked against the original before anything is published. These are Anthropic's own words; the bar
is 100% faithful.

```bash
node build.mjs            # generate any missing audio, render every experience + the library
node build.mjs <id>       # (re)build one work
node build.mjs --html-only # render HTML only (browser-TTS fallback, no API calls)
```

Narration audio is generated with the ElevenLabs API. Set `ELEVENLABS_API_KEY` and `VOICE_ID` in a
local `.env` (git-ignored — never commit it).
